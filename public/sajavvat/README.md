# Sajavvat photos

Drop real Sajavvat photos here. Until each file exists, the page falls back
to a tasteful Unsplash placeholder.

Expected filenames (so the page picks them up automatically):

| File              | Used on        | Aspect    | Suggested size |
|-------------------|----------------|-----------|----------------|
| `hero.jpg`        | Hero band      | 21 : 9    | 1600 × 685     |
| `envelopes.jpg`   | Envelopes card | 4 : 3     | 1200 × 900     |
| `decor.jpg`       | Decor card     | 4 : 3     | 1200 × 900     |
| `gifting.jpg`     | Gifting card   | 4 : 3     | 1200 × 900     |
| `gallery-1.jpg`   | Gallery slot 1 | 1 : 1     | 900 × 900      |
| `gallery-2.jpg`   | Gallery slot 2 | 1 : 1     | 900 × 900      |
| `gallery-3.jpg`   | Gallery slot 3 | 1 : 1     | 900 × 900      |
| `gallery-4.jpg`   | Gallery slot 4 | 1 : 1     | 900 × 900      |
| `gallery-5.jpg`   | Gallery slot 5 | 1 : 1     | 900 × 900      |
| `gallery-6.jpg`   | Gallery slot 6 | 1 : 1     | 900 × 900      |

JPGs only, sRGB colour space, quality ~80. Filenames are case-sensitive on
Vercel — keep them lowercase exactly as shown above.

To replace an image: just drop the new file in this folder with the same
name, commit, push. Vercel auto-deploys the new asset.
